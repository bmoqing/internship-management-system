# 高校学生实习管理平台

> 基于 Spring Boot + Vue 3 的前后端分离实习全生命周期管理系统

## 📋 项目简介

针对传统高校实习管理中信息不对称、过程监管缺失、评价体系单一等痛点，设计并开发了本系统。系统构建了**学生、教师、企业、管理员**四方协同生态，覆盖从岗位发布、双选审批、过程监督到多维评价的全链路闭环管理。

## 🧩 系统角色

| 角色 | 职责 |
|------|------|
| **管理员 (ADMIN)** | 用户管理、企业审核、系统配置、审计日志、全局监控 |

## ✨ 核心功能模块

### 1. 统一身份认证与权限控制
- 注册/登录/退出、令牌自动刷新
- **JWT 无状态认证** + Spring Security 过滤器链
- **RBAC 动态权限**：前端路由守卫 + 后端角色校验 + 数据层用户过滤 三层防线
- **防暴力破解**：登录失败次数缓存，超限 IP 临时封禁
- **图形验证码**：集成 Hutool 滑块验证码

### 2. 实习岗位双选与审批流转
- 企业发布岗位 → 学生投递 → **企业预审** → **教师审核** → **管理员终审**
- **状态机驱动**：待审核 → 待企业预审 → 待教师审核 → 已通过 / 已拒绝
- **防并发与防重复投递**：数据库层面唯一约束 + 事务声明
- 教师/企业端审批工作台，多状态筛选

### 3. 过程化监督与考勤管理
- **服务端时间戳防伪打卡**：基于 Leaflet 地图 + 服务器时间校验
- **定时预警任务**：缺席打卡 / 缺交日志主动告警推送
- **日志批阅**：学生提交富文本日志 → 教师评分 + 批改意见
- **水平防越权**：教师只能批阅自己所带学生的日志

### 4. 多维考核与成绩评价
- **加权聚合算法**：`总成绩 = 教师主观分 × W₁ + 企业评分 × W₂ + 日志均分 × W₃ + 考勤分 × W₄`
- 权重动态可配置（支持系统参数调整）
- 企业回传评分机制，打破传统单一评价模式

### 5. 申诉与复议
- 学生对成绩异议可发起申诉并上传举证材料
- 二级反向工作流：教师初审 → 管理员复议

### 6. 数据看板
- 基于 BFF 架构的千人千面仪表盘
- 按角色聚合数据定向分发
- 系统概览、待办事项、趋势统计

## 🚀 快速开始

### 环境要求

- **JDK** 17+
- **Maven** 3.9+
- **Node.js** 18+
- **MySQL** 8.0

### 1. 启动数据库

```bash
docker compose up -d
```

### 2. 启动后端

```bash
cd backend
# 配置 application.yml 中的数据库连接
mvn spring-boot:run
```

### 3. 启动前端

```bash
cd frontend
npm install
npm run dev
```

### 4. 访问系统

打开浏览器访问 `http://localhost:5173`

## 📊 数据库设计

系统包含 18+ 张核心数据表，覆盖用户、企业、岗位、申请、分配、考勤、日志、成绩、申诉、协议等全业务流程，采用第三范式（3NF）建模，主外键约束保障数据一致性。

主要数据表：`user`、`company`、`position`、`application`、`assignment`、`attendance`、`log`、`score`、`appeal`、`agreement`、`incident`、`sys_config`、`audit_log` 等。

## 🛡️ 安全设计

- **JWT 双令牌机制**：短时效 Access Token + 长时效 Refresh Token，自动续签
- **三层权限防线**：前端路由守卫 → 后端方法级权限注解 → 数据层用户身份过滤
- **水平防越权**：数据操作前强制校验师生/企业关联关系
- **登录保护**：BCrypt 密码加密 + 登录失败次数计数 + 黑名单封禁
- **令牌黑名单**：退出登录或修改密码后即时失效旧令牌
